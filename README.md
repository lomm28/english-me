## Installing app dependencies

### `npm install`

## Available Scripts

In the project directory, you can run:

### `npm run start`

Runs the webpack dev server in the development mode.<br>
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.

### `npm run build`

Builds the production bundle to dist directory.<br>

### `npm run lint`

Shows linting errors and warnings if any exist.<br>

### `npm run lint:fix`

Fixes linter errors.<br>
